Wer denkt sich die ganzen Standards aus?

W3C (World Wide Web Consortium)

  • Konsortium aus Unternehmen, Non-Profit-Organisationen,
    Universitäten, Behörden
  • Gründung 1994
  • Schaffung von Web-Standards

vs.

WHATWG (Web Hypertext Application Technology Working Group)

  • Arbeitsgruppe der Browserhersteller, u.a. Mozilla, Opera, Apple
  • Gründung 2004 auf Grund der schleppenden Arbeit des W3C
  • Versionslose HTML-Entwicklung & ergänzender Standards

=

  • XML-Revolution fällt aus: XHTML 2 Arbeitsgruppe 2009 eingestellt
  • Praxisnahe parallele bzw. gemeinsame Arbeit an HTML5

  • ... ist keine neue Technologie
  • ... erweitert bestehende Technogien

  • ... aktzeptiert proprietäre Technologien

  • ... erzeugt diese Präsentation!

Warum HTML5?

HTML 4

  • datiert von 1997
  • Fokus auf Web-Dokumenten

HTML5

  • implementiert stark vermisste Features nativ im Browser
  • ersetzt zahllose Hacks und Workarounds
  • PlugIn-freie Entwicklung
  • Fokus auf Web-Applikationen

 

HTML5

verwandelt den Browser vom Dokumentbetrachter zur universellen

Plattform für Applikationen

Einsetzbarkeit & Kompatibilität

  • Einige HTML5-Features komplett ausgereift und einsatzbereit
  • Andere Features nur teilweise oder gar nicht implementiert
  • Übergang von HTML 4 zu HTML5 möglich durch
    • Graceful Degradation, z.B. neue Formularfelder
    • Polyfills/Fallbacks, z.B. Video
    • Frameworks, z.B. Drag&Drop
  • Framework-Beispiele:

Geolocation

Standort des Benutzer bestimmen

navigator.geolocation.getCurrentPosition(function(position){
  position.timestamp;               // Zeitpunkt der Ortsbestimmung
  position.coords.latitude;         // Geographische Breite
  position.coords.longitude;        // Geographische Länge
  position.coords.altitude;         // Höhe
  position.coords.accuracy;         // Präzision Breite/Länge
  position.coords.accuracyAltitude; // Präzision Höhe
  position.coords.heading;          // Richtung
  position.coords.speed;            // Geschwindigkeit
});
